# PROJECT KNOWLEDGE — The Hunger Games: Dead Serious

> **Master Context Anchor** for AI collaborators reading this repo.

## Project Overview

- **Title**: The Hunger Games: Dead Serious (90-Second Parody)
- **Visual Anchor**: *The Hunger Games* (2012) — Cornucopia Bloodbath Sequence
- **Format**: 90-second AI-generated video assembled from Kling 3.0 clips, ElevenLabs voiceover, and Invideo timeline editing
- **Tone**: Dead serious. No comedy. No levity. Grounded survival horror played completely straight with absurd casting.

## Core Environmental Logic

| Property | Value |
|----------|-------|
| **Location** | High-altitude Appalachian Meadow / North Carolina Pine Forest |
| **Terrain** | Damp, dark gray earth; mud-slicked paths; tall, dead, yellow-brown grass |
| **Flora** | Dense, spindly pine and oak trees; overgrown, unmanaged wilderness |
| **Weather** | Constant flat overcast lighting; thick mountain mist; cold, damp atmosphere |

## Technical Cinematography (Kling AI Specs)

| Property | Value |
|----------|-------|
| **Film Stock** | Super 35mm film grit; visible grain; no "digital clean" look |
| **Color Grade** | Desaturated Bleach-Bypass; crushed blacks; teal-tinted shadows; muted earthy greens |
| **Motion** | Handheld "shaky-cam" documentary style; 45-degree shutter angle (crisp, jittery, panicked motion) |
| **Framing** | 2.35:1 Anamorphic; claustrophobic close-ups mixed with wide desolate meadow shots |
| **FPS** | 24fps |

## Character Performance Directives

- **Emotional State**: High-anxiety; pure survival instinct; terrified and exhausted
- **Wardrobe Texture**: Dusty tactical gear; visible sweat and grime; torn fabrics

### Hard Negative Constraints

```
NO smiling
NO vibrant/saturated colors
NO heroic sunlight
NO clean skin
NO digital-clean rendering
```

## Arena Geography

- **Cornucopia**: Golden horn-shaped structure at arena center, supplies scattered on grass
- **Launch Platforms**: Circular stone platforms with metal tribute tubes (glass cylinders rising from underground)
- **Treeline**: Dense forest 200m from Cornucopia ring
- **Arena Floor**: Packed earth and grass — NOT mud or trenches

## Production Pipeline

1. **Dashboard** (this app) → Character management, prompt generation, style bible
2. **Kling 3.0** → AI video generation from prompts
3. **ElevenLabs** → Voiceover / narrator TTS
4. **Invideo** → Timeline assembly, SFX, final edit
5. **Make.com** → Webhook automation for data sync
